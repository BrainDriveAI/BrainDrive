# Relationships Journal - Agent Context

*Folder scope for owner-entered Relationships follow-up history.*

This folder stores the owner's Relationships journal after `spec.md` and `plan.md` exist. The journal is owner-driven follow-up context, not a CRM, tracker, or passive memory feed.

## Preservation Rule

Before writing, correcting, or recovering `journal.md`, read the existing file when it is readable. Never replace the whole file. Append new entries or targeted-edit only the intended entry. If `journal.md` is unreadable or corrupt, create a timestamped recovery file, preserve the original, and tell the owner what happened.

## What Belongs Here

- Owner-provided relationship events, reflections, conversations, follow-ups, commitments, and context changes.
- Sensitive third-party context only at useful owner-provided detail, with uncertainty preserved.
- Recurring situations, boundary patterns, communication blockers, follow-up gaps, and owner-controlled next steps.
- Concrete commitments in `Follow-up -> Proposed task` for V.1.

## What Does Not Belong Here

- Workflow instructions, prompts, rubrics, reports, source files, or test artifacts.
- Raw copies of Relationships `spec.md`, Relationships `plan.md`, `me/profile.md`, or task lists.
- Invented conversations, feelings, motives, facts, diagnoses, or commitments about other people.
- CRM, dating, therapy, legal, mediation, crisis, passive ingestion, automatic outreach, surveillance, or manipulation state.

## Parent And Profile Updates

Relationships `spec.md` owns current goals, owner-provided context, boundaries, assumptions, and success criteria. Relationships `plan.md` owns ordered next steps, conversation prep, milestones, and plan status. Update those parent files only after the owner agrees to a concise current-truth change. Stable cross-project owner facts go to `me/profile.md` only after confirmation.

## Professional-Care Boundary

When needed, use:

> I can help you reflect on what happened, notice patterns, prepare for conversations, and decide what you want to do next. I cannot replace therapy, legal advice, mediation, crisis support, or professional care. If this involves safety, abuse, legal risk, or urgent distress, consider contacting an appropriate professional or local emergency resource.
